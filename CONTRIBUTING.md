# Contributing to 555 Timer Lab

We welcome contributions! If you'd like to improve this project, please follow these guidelines.

## How to Contribute

1. **Fork the repository**: Click the "Fork" button on GitHub.
2. **Clone your fork**:
   ```bash
   git clone https://github.com/yourusername/555-timer-lab.git
   ```
3. **Create a branch**:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make changes**: Implement your feature or fix.
5. **Test**: Ensure the app runs correctly (`npm run dev`).
6. **Commit**: Write clear commit messages.
   ```bash
   git commit -m "Add feature: ..."
   ```
7. **Push**:
   ```bash
   git push origin feature/your-feature-name
   ```
8. **Create a Pull Request**: Open a PR on the original repository.

## Code Style

- Use TypeScript for type safety.
- Follow the existing project structure (components, utils, etc.).
- Use Tailwind CSS for styling.
- Keep components functional and use hooks where possible.

## Issues

If you find a bug or have a feature request, please open an issue on GitHub.
